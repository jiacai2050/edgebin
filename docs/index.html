httpbin

A simple HTTP Request & Response Service

Similar to httpbin.org but built with Cloudflare Workers.

A hosted version is available for free: https://httpbin.liujiacai.net
You can also deploy your own instance with the button below:

Available Endpoints

  1. /get
    Returns GET data. 
  2. /ip
    Returns your IP address. 
  3. /headers
    Returns request headers. 
  4. /user-agent
    Returns your User-Agent string. 
  5. /anything
    Returns anything you send in the request (GET, POST, etc.). 
  6. /delay?seconds=:number
    Delays the response by the specified number of seconds (max 10). 
  7. /redirect-to?url=:new-url
    Redirects to the specified URL. 
  8. /redirect/:number
    Redirects N times before landing on /get. 
  9. /response-headers?freeform=123&another=456
    Returns the headers you specify as query parameters. 
  10. /status/:code
    Returns a response with the specified status code. 
  11. /cookies
    Returns the cookies you sent.

For /anything, you can send various types of data including JSON, form data, and files.
